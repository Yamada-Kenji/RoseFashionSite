ROSE FASHION WEBSITE

SETUP
Database: 
 - Create database in Microsoft SQL Server with .bak file in "database" folder (Latest file: db-14-7.bak)

Backend:
 - Open "RoseFashionBE" folder in Visual Studio then run the project with Ctrl + F5

Front-end:
 - Setting up the local environment and workspace following this site: https://angular.io/guide/setup-local
 - Open "RoseFashionFE" folder in Visual Studio Code then run the command: ng server -o
